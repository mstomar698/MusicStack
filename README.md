## MusicStack

- A Simple app that works on paid APi to provide music data to the user.
- The is made using React, Redux And Tailwind CSS.
- The APP is Deployed to Netlify.

### To run locally

```bash
git clone https://github.com/mstomar698/MusicStack.git
cd MusicStack
npm install
npm run dev
```
